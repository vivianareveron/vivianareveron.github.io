JavaScript Project

An interactive module for a photographic portfolio

For this project we were asked to make a simple program using JavaScript. I had creative freedom to make any program, but there were rules about which instructions to use. I decided to make a module that allows you to play with pictures, when you hover on one of the parts it changes randomly to a new picture, and if you want to see the full image, just click on it.
